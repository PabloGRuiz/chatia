# Documentos de Ejemplo

Guarda en esta carpeta los documentos (.pdf, .docx, .txt) que desees llevar a la oficina o utilizar para pruebas en la base de datos de Chat IA. 

Al estar en esta ruta, Git **no** los ignorará, permitiendo sincronizarlos a través del repositorio de forma segura para tenerlos a disposición en cualquier equipo.

*Nota: Una vez en la oficina, puedes subir estos archivos al sistema desde el panel de Administración.*
